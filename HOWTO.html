Radify Menu Editor Modular Edition

AutoHotkey v2 用のリングメニューライブラリ "Radify" の設定を、直感的なGUIで作成・編集するためのツールです。
複雑なJSONや配列を手書きする必要なく、ドラッグ＆ドロップでメニューを構築し、すぐに実行可能なスクリプトを生成できます。

🚀 導入と起動

必須環境

  • Windows 10 / 11
  • Python 3.x (CustomTkinter等のライブラリが必要)
  • AutoHotkey v2.0+ (生成されたスクリプトの実行に必要)

起動方法

AHK_Radify_Editor_Modular フォルダ内の main.py を実行してください。

ヒント: 初回起動時、必要な設定ファイル (menu_config.json) がない場合は自動的に作成されます。

📋 基本操作

画面構成

  • 左パネル (メニュー構造):
    ツリー形式でメニュー（リング、サブメニュー）を管理します。ドラッグ＆ドロップでアイテムの並び替えが可能です。
  • 中央パネル (プレビュー):
    実際のメニューの見た目をリアルタイムで確認できます。ズーム操作も可能です。
  • 右パネル (エディタ):
    選択したアイテムの詳細（テキスト、アイコン、動作）を編集します。

ショートカットキー

  • Ctrl + S : 設定の上書き保存
  • Ctrl + Z : 元に戻す (Undo)
  • Ctrl + Y : やり直し (Redo)
  • Mouse Wheel : プレビューのズーム (中央パネル上)

構造の編集

左側のボタンを使って構造を変更します。

  • ➕ リング追加 : メインメニューまたはサブメニューに新しいリング（円）を追加します。
  • ➕ アイテム追加 : 選択中のリングに新しいボタンを追加します。
  • 🗑️ 削除 : 選択したアイテムやリングを削除します。複数選択も可能です。
ポイント: ツリー上でアイテムをドラッグ＆ドロップすると、リング間やサブメニュー間での移動が簡単に行えます。

✏️ アイテム編集

アイテムを選択すると右側のパネルで詳細を設定できます。変更は即座にプレビューに反映されます (Live Sync)。

1. アイコン設定 (🖼️)

メニューに表示する画像を設定します。

  • ドロップダウン: images フォルダにある画像から選択。
  • 📁 追加: PC内の画像ファイルを取り込みます。
  • 📷 取り込み: 画面キャプチャ機能です。範囲選択してアイコンとして保存できます。
  • ⚙️ 管理: 不要な画像の削除やリネームを行えます。

2. クリック動作 (🖱️)

アイテムをクリックしたときに実行される AutoHotkey v2 コードを記述します。

基本パターン

; URLを開く
Run("https://google.com")

; アプリを起動
Run("notepad.exe")

; キー送信 (Ctrl+C)
Send("^c")

高度な記述 (複数行対応)

if 文などの複雑なロジックも記述可能です。エディタが自動的に (*) => { ... } ブロックとして処理します。 

if WinActive("ahk_exe chrome.exe") {
    Send("^t") ; 新しいタブ
} else {
    Run("chrome.exe")
}
便利機能: 🔧 カスタム ボタンを押すと、よく使うコードのテンプレートライブラリが開きます。検索してワンクリックで挿入できます。

✨ 便利機能

★ お気に入り機能

よく使うアイテム設定（アイコンとコマンドのセットなど）を「お気に入り」として保存できます。 別のアイテムを選択して適用することで、瞬時に設定をコピーできます。

🔄 バックアップと復元

保存のたびに自動的にバックアップが作成されます（最新10件）。 間違えて保存してしまっても、📂 バックアップ復元 から過去の状態に戻せます。

📊 統計情報

メニュー全体のアイテム数や画像数を確認できます。大規模なメニューを作るときの管理に役立ちます。

📂 サブメニューの活用

アイテム数が多くなった場合は、サブメニュー（階層メニュー）を使用します。

作成手順

  1. 左パネルの ➕ サブメニュー追加 で新しいサブメニュー枠を作成します。
  2. 作成されたサブメニューにリングやアイテムを追加して編集します。
  3. メインメニュー（親）に戻り、サブメニューを呼び出したいアイテムを選択します。
  4. 右パネルの 「📂 サブメニュー設定:」 ドロップダウンから、作成したサブメニューを選択します。
これにより submenu プロパティが設定され、Radifyが自動的にサブメニュー展開を処理します。
※ click 動作に Radify.Show(...) を書く古い方法も可能ですが、ドロップダウンからの設定（推奨）の方が高速です。

💾 生成と保存

AHKスクリプトの生成

💾 保存＆AHKコード生成 ボタンを押すと、現在の設定に基づいた .ahk ファイルが出力されます。 生成されたファイルを実行するだけで、リングメニューが使用可能になります。

単体パッケージ出力 (配布用)

画面下部の 「📦 単体パッケージとして出力」 にチェックを入れて生成すると、以下のファイルがセットで出力されます：

  • 生成された .ahk スクリプト
  • Lib フォルダ (Gdipなどの必須ライブラリ)
  • images フォルダ (使用しているアイコン画像)
  • Radify.ahk (コアライブラリ)
この機能を使えば、生成されたフォルダをそのまま別のPCにコピーするだけで動作します。
