# 基本操作

このページでは、Radify Menu Editorの基本的な操作方法を説明します。

## UI構成

### メニューバー

- **ファイル**: 保存、読込、新規作成、エクスポート
- **編集**: Undo/Redo、コピー&ペースト
- **表示**: プレビューのズーム、テーマ切り替え
- **ヘルプ**: ドキュメント、バージョン情報

### ツールバー

左パネルの上部に配置されている主要な操作ボタン:

- **新規リング追加**: 新しいリングを追加
- **新規アイテム追加**: 選択中のリングにアイテムを追加
- **サブメニュー追加**: サブメニューを作成
- **削除**: 選択中の要素を削除
- **上へ/下へ**: アイテムの順序を変更

## リングの管理

### リングの追加

1. 「新規リング追加」ボタンをクリック
2. リングがツリービューに追加される
3. プレビューに円形の配置が表示される

### リングの設定

リングを選択すると、右パネルで以下を設定できます:

- **リング番号**: リングの識別番号（内側から1, 2, 3...）
- **半径**: 中心からの距離
- **分割数**: 円周上のスロット数

!!! tip "リング半径の調整"
    半径を調整することで、多重リング構造を作成できます。

### リングの削除

1. 削除したいリングを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「はい」を選択

!!! warning "注意"
    リングを削除すると、そのリングに含まれるすべてのアイテムも削除されます。

## アイテムの管理

### アイテムの追加

1. アイテムを追加したいリングを選択
2. 「新規アイテム追加」ボタンをクリック
3. 新しいアイテムがリングに追加される

### アイテムの選択

- ツリービューで直接クリック
- プレビューキャンバス上でクリック（実装されている場合）

### アイテムの並び替え

#### 方法1: ボタンを使用

1. 移動したいアイテムを選択
2. 「上へ」または「下へ」ボタンをクリック

#### 方法2: ドラッグ&ドロップ

1. アイテムをツリービューでドラッグ
2. 目的の位置でドロップ

!!! tip "視覚的な並び替え"
    ドラッグ&ドロップが最も直感的で、リアルタイムでプレビューに反映されます。

## プレビュー機能

### ズーム

プレビューのサイズを調整できます:

- **拡大**: `Ctrl` + `+`
- **縮小**: `Ctrl` + `-`
- **リセット**: `Ctrl` + `0`

またはプレビューパネルのズームスライダーを使用します。

### ホバー表示

プレビュー上のアイテムにマウスを乗せると、そのアイテムの情報が表示されます。

## コピー&ペースト

### アイテムのコピー

1. コピーしたいアイテムを選択
2. `Ctrl+C` または「編集」→「コピー」

### アイテムのペースト

1. ペースト先のリングを選択
2. `Ctrl+V` または「編集」→「ペースト」

!!! info "複製"
    同じリング内でペーストすると、アイテムが複製されます。

## Undo/Redo

編集履歴を管理できます:

- **元に戻す**: `Ctrl+Z`
- **やり直す**: `Ctrl+Y`

!!! note "履歴の保持"
    履歴は最大50ステップまで保持されます。

## 検索機能

### アイテムの検索

右パネル上部の検索ボックスに文字を入力すると、ラベルやアクションに含まれるアイテムを検索できます。

### 画像の検索

画像リストタブで、画像ファイル名で検索できます。

## キーボードショートカット

| ショートカット | 機能 |
|---|---|
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新規プロジェクト |
| `Ctrl+O` | ファイルを開く |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` | やり直す |
| `Ctrl+C` | コピー |
| `Ctrl+V` | ペースト |
| `Delete` | 削除 |
| `F5` | プレビューを更新 |
