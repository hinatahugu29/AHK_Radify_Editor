# アイコンとビジュアル

このページでは、メニューの見た目をカスタマイズする方法を説明します。

## アイコンの設定

### アイコンの選択方法

#### 方法1: ファイルから選択

1. アイテムを選択
2. 右パネルの「アイコン選択」をクリック
3. 画像ファイル（PNG, JPG, ICO）を選択

!!! tip "対応形式"
    PNG, JPG, JPEG, BMP, ICO, GIF形式に対応しています。

#### 方法2: 画面キャプチャ

**Radify Menu Editorの最大の特徴**である画面キャプチャ機能:

1. 「画面キャプチャ」ボタンをクリック
2. エディタが最小化される
3. マウスドラッグで範囲を選択
4. 自動的に切り抜き・リサイズされてアイコンに設定される

!!! success "便利な使い方"
    アプリケーションのアイコンやボタンを直接取り込めるので、一目でわかるメニューが作れます。

#### 方法3: 既存の画像リストから選択

1. 右パネルの「画像リスト」タブを開く
2. 使用したい画像をクリック
3. 「この画像を使用」をクリック

### アイコンのサイズ

アイコンは自動的に最適なサイズにリサイズされます。

- デフォルト: 64x64px
- 推奨サイズ: 32x32 〜 128x128px

!!! note "透過PNG推奨"
    背景が透明なPNG画像を使用すると、きれいに表示されます。

## ラベルの設定

### ラベルテキスト

右パネルの「ラベル」フィールドに表示したいテキストを入力します。

```
例: メモ帳
例: Google検索
例: 音量↑
```

### ラベルの表示/非表示

- アイコンのみ表示
- ラベルのみ表示
- 両方表示（デフォルト）

設定は右パネルのチェックボックスで切り替えられます。

### フォント設定

メニュー全体のフォント設定:

1. 「メニューオプション」を開く
2. フォントファミリーを選択
3. フォントサイズを調整

```
推奨フォント:
- Yu Gothic UI
- Meiryo UI
- Segoe UI
```

## カラーテーマ

### スキンの選択

右パネルの「メニューオプション」でスキンを選択できます:

- **Default**: 標準の配色
- **Dark**: ダークテーマ
- **Light**: ライトテーマ
- **Custom**: カスタマイズ可能

### グロー効果

アイテムのグロー（光る効果）を有効/無効にできます。

- **Enable Glow**: チェックで有効化
- グローの色はスキンに依存

!!! tip "視認性向上"
    グロー効果を有効にすると、暗い背景でもアイテムが見やすくなります。

## プレビューのカスタマイズ

### ズームレベル

プレビューの表示倍率を調整:

- スライダーで調整: 50% 〜 200%
- ショートカット:
  - `Ctrl +`: 拡大
  - `Ctrl -`: 縮小
  - `Ctrl 0`: リセット

### 背景色

プレビューの背景色を変更して、実際の使用環境に近い色で確認できます。

## メニュー全体の設定

### メニューオプション

「メニューオプション」ダイアログで以下を設定:

#### アイテムサイズ

```
itemSize: 60
```

アイテムアイコンの基本サイズ（ピクセル単位）

#### リング間隔

```
ringSpacing: 80
```

リング間の距離（ピクセル単位）

#### メニュー名

```
menu_name: myMainMenu
```

AutoHotkeyスクリプト内で使用される変数名

### 外観モード

- **Light**: ライトモード（明るい背景）
- **Dark**: ダークモード（暗い背景）
- **System**: システム設定に従う

## アイコンリソースの管理

### 画像の整理

`images/` フォルダ内の画像は以下のように整理することをお勧めします:

```
images/
├── apps/          # アプリケーションアイコン
├── system/        # システム機能アイコン
├── custom/        # カスタムアイコン
└── emoji/         # 絵文字アイコン
```

### 未使用画像の削除

エディタには未使用の画像を検出する機能があります（将来実装予定）。

現在は手動で確認:

1. 画像リストタブを開く
2. 各画像の使用状況を確認
3. 未使用の画像を削除

## ビジュアルのベストプラクティス

### 一貫性のあるデザイン

- 同じスタイルのアイコンを使用
- カラースキームを統一
- サイズを統一（自動でリサイズされるが、元画像も近いサイズに）

### 視認性の確保

- 背景とのコントラストを意識
- 小さすぎるアイコンは避ける
- ラベルは短く、わかりやすく

### テーマ別のアイコンセット

機能ごとにアイコンのテーマを揃える:

- **ファイル操作**: フォルダ、ドキュメントアイコン
- **Web関連**: ブラウザ、SNSアイコン
- **システム**: 歯車、設定アイコン
- **メディア**: 再生、停止アイコン

## カスタムアイコンの作成

### オンラインアイコンジェネレーター

以下のサービスで無料アイコンを作成・ダウンロードできます:

- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
- [Font Awesome](https://fontawesome.com/)

### アイコン編集ソフト

- **GIMP**: 無料の画像編集ソフト
- **Paint.NET**: Windowsで使いやすい
- **Photoshop**: プロ向け

### 推奨設定

アイコンを作成する際の推奨設定:

- **形式**: PNG（透過背景）
- **サイズ**: 64x64px または 128x128px
- **色深度**: 32bit（透明度あり）

## トラブルシューティング

### アイコンが表示されない

1. 画像ファイルが存在するか確認
2. ファイル形式が対応しているか確認（PNG, JPG等）
3. 画像が破損していないか確認

### アイコンがぼやける

- より高解像度の画像を使用
- 元画像のサイズが小さすぎる可能性

### 画面キャプチャが動作しない

- エディタを管理者として実行してみる
- セキュリティソフトが干渉していないか確認

## サンプルアイコンセット

エディタに同梱されているアイコンセット（`images/`フォルダ）:

- システムアプリケーション（50種類以上）
- 絵文字アイコン（100種類以上）
- Web サービス（30種類以上）

!!! tip "アイコンの再利用"
    他のプロジェクトで使用したアイコンは、`images/`フォルダをコピーすることで再利用できます。
