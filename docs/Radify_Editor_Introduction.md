## 【自作ソフト】AHK v2製リングメニューを完全GUI化。「Radify Menu Editor」でPC操作を“直感”に変える

AutoHotkey v2はその柔軟性ゆえに、GUIランチャーやリングメニューを作ろうとすると「座標計算」や「複雑な配列定義」という壁に直面しがちです。洗練されたリングメニューライブラリ **"Radify"** を、1行のコードも書かずに、しかし必要なロジックはフルスクラッチ並みに詳細に設定できる――そんな理想の開発環境として **「Radify Menu Editor」** を開発しました。

ただのランチャー設定ツールではありません。これはあなたのPC操作を「視覚的」かつ「論理的」に再構築するための **統合開発環境（IDE）** です。

## 開発の動機と解決策：テキスト編集からの解放

従来のリングメニュー作成は、JSONファイルやAHKスクリプトを直接編集する「手探りの作業」でした。「アイコンのパスは合っているか？」「サブメニューの階層はずれていないか？」といった本質的でない確認作業に時間を奪われるのが常でした。

**Radify Menu Editor** は、このプロセスを完全にGUI化しました。左パネルのツリービューで構造を把握し、中央のキャンバスでデザインをリアルタイムに確認し、右パネルでロジックを注入する。この「構造・デザイン・ロジック」の三位一体を、シームレスなUIで実現しています。

## 徹底した「効率」へのこだわりと機能詳細

本ツールの機能は、単にメニューが作れるだけにとどまりません。実際にメニューを「運用」し、「修正」し続けるユーザーの視点に立った機能セットを搭載しています。

###1. 「画面そのもの」をアイコンにする瞬発力

リングメニューの視認性を高める鍵はアイコンですが、適切な画像ファイルを探してリサイズしてフォルダに保存する作業は苦痛です。本ツールには **「画面キャプチャ取り込み機能」** を標準搭載しました。

エディタ上のボタン一つで画面の一部を切り抜き、自動的にリサイズしてアイコンとして登録・適用します。「このソフトのこのボタン」をそのままメニューアイコンにする。この直感的なワークフローが、メニュー構築のハードルを劇的に下げます。

### 2. No-CodeとPro-Codeの融合：柔軟なアクション記述

アクション定義においては、初心者向けの「テンプレート」と、上級者向けの「フルスクリプト記述」の両立を目指しました。
一般的なランチャー設定にある「パスを指定して終わり」ではなく、本ツールは **AutoHotkey v2のコードを直接記述可能** です。

`Run`関数はもちろん、`if WinActive(...)` による条件分岐や、変数操作などの複雑なロジックも、エディタが自動的に無名関数 `(*) => { ... }` としてラップするため、制約なく記述できます。一方で、よく使う「アプリ起動」や「キー送信」は、内蔵テンプレートから検索してワンクリックで挿入可能。手軽さと拡張性のバランスを極限まで高めました。

### 3. "Live Sync" と堅牢なデータ管理

右パネルで編集したパラメータは、即座に中央のプレビューに反映されます（**Live Sync**）。文字サイズ、アイコンの配置、円の分割数などを試行錯誤する際に、再読み込みの待ち時間はゼロです。

また、編集データは堅牢なJSON形式で管理され、保存時には自動的にバックアップ（履歴保持）が作成されます。`Ctrl+Z` / `Ctrl+Y` によるUndo/Redoも完備しており、細部までこだわり抜いたメニュー作りをサポートします。

### 4. 配布・ポータビリティへの配慮

作成したメニューは自分だけで使うのはもったいないかもしれません。エディタの **「単体パッケージとして出力（Export）」** 機能を使用すれば、生成されたAHKスクリプトだけでなく、必要なライブラリ群（Lib）や使用している画像リソース（images）を構造化して一つのフォルダにまとめます。

このフォルダをUSBメモリに入れたりクラウドで共有したりすれば、AHK v2がインストールされているどのWindows環境でも、あなたの作り込んだリングメニューを即座に再現可能です。

## 導入方法と動作環境

Windows 10/11環境にて、Python 3.x（エディタ起動用）およびAutoHotkey v2（生成スクリプト実行用）が必要です。
`AHK_Radify_Editor_Modular` フォルダ内の `main.py` を起動するだけで、あなたのデスクトップは新しい操作体系へと進化します。

「Radify Menu Editor」は、単なる時短ツールではなく、PC操作という体験そのものをデザインするためのプラットフォームです。無機質なキーボードショートカットの世界から、色彩豊かで直感的なリングメニューの世界へ。ぜひ、この快適さを体験してください。
