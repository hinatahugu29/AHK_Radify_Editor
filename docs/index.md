# Radify Menu Editor ドキュメント

![Radify Menu Editor](../images/radify-skin-editor.png)

## ようこそ

**Radify Menu Editor** へようこそ！このドキュメントでは、AutoHotkey v2製リングメニューを完全GUI化する統合開発環境の使い方を詳しく解説します。

## このツールについて

AutoHotkey v2はその柔軟性ゆえに、GUIランチャーやリングメニューを作ろうとすると「座標計算」や「複雑な配列定義」という壁に直面しがちです。洗練されたリングメニューライブラリ **"Radify"** を、1行のコードも書かずに、しかし必要なロジックはフルスクラッチ並みに詳細に設定できる――そんな理想の開発環境として **Radify Menu Editor** を開発しました。

## 🌟 主な特徴

### 完全GUI化
テキストエディタを開く必要はありません。すべての設定をビジュアルインターフェースで完結できます。

### リアルタイムプレビュー
編集した内容は即座に中央のキャンバスに反映されます（**Live Sync**）。文字サイズ、アイコンの配置、円の分割数などを試行錯誤する際に、再読み込みの待ち時間はゼロです。

### 柔軟なアクション定義
初心者向けの「テンプレート」と、上級者向けの「フルスクリプト記述」の両立を実現しました。AutoHotkey v2のコードを直接記述可能で、`if WinActive(...)` による条件分岐や変数操作などの複雑なロジックも制約なく記述できます。

### 画面キャプチャ統合
エディタ上のボタン一つで画面の一部を切り抜き、自動的にリサイズしてアイコンとして登録・適用します。「このソフトのこのボタン」をそのままメニューアイコンにできる、この直感的なワークフローが開発のハードルを劇的に下げます。

### ドラッグ&ドロップ
ツリービューで直感的にメニュー構造を編集できます。アイテムの並び替えや階層の変更が簡単に行えます。

### Undo/Redo対応
`Ctrl+Z` / `Ctrl+Y` による編集履歴管理を完備しており、細部までこだわり抜いたメニュー作りをサポートします。

### 自動バックアップ
編集データは堅牢なJSON形式で管理され、保存時には自動的にバックアップ（履歴保持）が作成されます。

### ポータブル出力
作成したメニューは **「単体パッケージとして出力（Export）」** 機能を使用することで、生成されたAHKスクリプトだけでなく、必要なライブラリ群（Lib）や使用している画像リソース（images）を構造化して一つのフォルダにまとめることができます。
