# クイックスタート

このガイドでは、Radify Menu Editorを使って最初のリングメニューを作成する方法を説明します。

## エディタの起動

プロジェクトフォルダで以下のコマンドを実行します:

```bash
python main.py
```

## 画面構成の理解

エディタは3つの主要なパネルで構成されています:

### 左パネル: ツリービュー
- メニューの階層構造を表示
- ドラッグ&ドロップで並び替え可能

### 中央パネル: プレビュー
- リアルタイムでメニューの見た目を確認
- Live Sync機能により即座に反映

### 右パネル: プロパティエディタ
- 選択したアイテムの詳細設定
- アイコン、ラベル、アクションを編集

## 最初のメニューを作成

### Step 1: 新規リングを追加

1. 左パネル上部の **「新規リング追加」** ボタンをクリック
2. リングが追加され、ツリーに表示されます

!!! tip "リングとは？"
    リングは、アイテムを円形に配置するための単位です。メニューは複数のリングで構成できます。

### Step 2: アイテムを追加

1. 追加したリングを選択
2. **「新規アイテム追加」** ボタンをクリック
3. アイテムがリングに追加されます

### Step 3: アイテムの設定

右パネルで以下を設定します:

#### ラベル
アイテムに表示されるテキストを入力します。

```
例: メモ帳
```

#### アイコン
1. 「アイコン選択」をクリック
2. 画像ファイルを選択、または
3. 「画面キャプチャ」で画面から直接取り込み

!!! tip "画面キャプチャ"
    画面キャプチャ機能を使うと、アプリケーションのアイコンやボタンを直接取り込めます。

#### アクション
アイテムをクリックしたときの動作を設定します。

##### 方法1: テンプレートを使用

1. 「テンプレート検索」ボタンをクリック
2. 「アプリ起動」などのテンプレートを選択
3. パラメータを入力（例: `notepad.exe`）

##### 方法2: 直接コードを記述

上級者向け。AutoHotkey v2のコードを直接記述できます:

```ahk
Run "notepad.exe"
```

### Step 4: プレビューで確認

中央のキャンバスに、作成したメニューがリアルタイムで表示されます。

- アイコンの配置
- 文字サイズ
- 色とスタイル

すべて即座に反映されます。

### Step 5: 保存

1. メニューバーから **「ファイル」→「保存」**
2. または `Ctrl+S` キーを押す

!!! info "自動保存"
    エディタは定期的に自動保存を行いますが、手動保存も推奨します。

## AHKスクリプトとして出力

作成したメニューをAutoHotkeyスクリプトとして出力できます。

1. メニューバーから **「ファイル」→「AHKスクリプト出力」**
2. 出力先フォルダを選択
3. 必要なファイルが自動的にコピーされます:
   - メインスクリプト（.ahk）
   - ライブラリファイル（Lib/）
   - 画像リソース（images/）

### スクリプトの実行

出力されたフォルダ内の `.ahk` ファイルをダブルクリックするだけで実行できます。

!!! success "完成！"
    これであなたの最初のリングメニューが完成しました！

## よくある質問

### Q: 作成したメニューが表示されない

A: AutoHotkey v2がインストールされていることを確認してください。また、スクリプトを右クリック→「管理者として実行」を試してください。

### Q: アイコンが表示されない

A: 画像ファイルのパスが正しいか確認してください。相対パスを使用している場合は、スクリプトと同じフォルダに画像を配置してください。

### Q: アクションが実行されない

A: アクションコードに構文エラーがないか確認してください。テンプレートから始めることをお勧めします。
